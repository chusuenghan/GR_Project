# GR_Project
# Spring 기반 매칭 웹 서비스

게시판 기반 거래 매칭 웹 서비스입니다.  
사용자 간 거래 요청, 실시간 채팅, 거래 완료 후 평가 기능을 제공합니다.

## Tech Stack
- Spring MVC
- MyBatis
- MariaDB
- WebSocket (STOMP, SockJS)
- JSP

## Main Features
- 게시글 CRUD 및 댓글
- 거래 요청 및 상태 관리
- WebSocket 기반 1:1 실시간 채팅
- 거래 종료 후 평가 및 평점 반영

## Project Overview
- 개인 프로젝트
- 거래 흐름: 게시글 → 요청 → 채팅 → 거래 종료 → 평가
